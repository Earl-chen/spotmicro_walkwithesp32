# 四足机器狗运动学测试程序

## 概述

本程序是一个专门用于测试四足机器狗正/逆运动学（FK/IK）正确性的命令行程序。它基于C++实现，与Python参考实现保持算法一致性，提供高精度的运动学计算验证。

## 功能特性

### 🧪 测试类型
- **FK-IK往返测试**：验证正向和逆向运动学的计算精度
- **批量预定义测试**：执行预设的测试用例集
- **交互式测试**：支持手动输入参数进行实时测试
- **快速验证测试**：单个典型用例的快速检验

### 📊 测试覆盖
- **四条腿测试**：左前、右前、左后、右后腿
- **多种姿态**：零位、典型姿态、极限姿态、小角度等
- **高精度验证**：角度误差通常在微弧度级别（< 0.001°）

### 📈 报告功能
- **实时结果显示**：测试过程中实时显示结果
- **详细统计信息**：成功率、误差分布、执行时间等
- **报告保存**：支持保存为文本文件，包含完整测试详情

## 编译和运行

### 编译
```bash
# 使用提供的Makefile编译
make -f Makefile.kinematics_test

# 清理编译文件
make -f Makefile.kinematics_test clean
```

### 运行选项
```bash
# 快速验证测试
./kinematics_test --quick
./kinematics_test -q

# FK-IK往返测试
./kinematics_test --roundtrip
./kinematics_test -r

# 批量预定义测试
./kinematics_test --batch
./kinematics_test -b

# 交互式菜单模式
./kinematics_test

# 显示帮助信息
./kinematics_test --help
./kinematics_test -h
```

## 技术特性

### 算法一致性
- 与Python参考实现完全一致的数学算法
- 修复了原始C++实现中的关键差异：
  - 右腿髋侧摆角度计算公式
  - 髋俯仰角度计算顺序
  - 膝俯仰角度符号处理

### 精度保证
- 使用单精度浮点数计算
- 实施数值稳定性保护（角度范围约束）
- 微弧度级别的计算精度

### 几何参数
- L1 = 60.5mm（髋关节延伸段）
- L2 = 10mm（髋关节垂直段）
- L3 = 111.126mm（大腿长度）
- L4 = 118.5mm（小腿长度）
