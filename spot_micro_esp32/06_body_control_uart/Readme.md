## 命令接口

### 模式切换
- `pwm` - PWM直接控制模式
- `angle` - 舵机角度控制模式
- `joint` - 关节角度控制模式
- `kinematics` - 运动学控制模式

### 控制命令
- `s <编号>` - 选择舵机 (0-11)
- `<数值>` - 控制当前选中的舵机
- `<舵机ID> <值>` - 直接控制指定舵机
- `j <腿> <关节> <角度>` - 控制特定关节
  - 腿: 0=左前, 1=右前, 2=左后, 3=右后
  - 关节: 0=髋侧摆, 1=髋俯仰, 2=膝俯仰

### 预设姿势
- `sleep0` - 趴下姿势
- `sleep1` - 站立姿势

### 系统命令
- `help` - 显示帮助信息
- `status` - 显示系统状态
