# ESP32 四足机器狗硬件连接指南

## 1. 核心引脚连接

ESP32 通过 **I2C** 接口连接 **PCA9685** PWM驱动芯片来控制12个舵机：

| 引脚 | GPIO | 功能 | 说明 |
|------|------|------|------|
| **SDA** | **GPIO21** | I2C 数据线 | 连接 PCA9685 SDA |
| **SCL** | **GPIO22** | I2C 时钟线 | 连接 PCA9685 SCL |

---

## 3. 代码中的配置

### PCA9685Driver.hpp (第64行)
```cpp
PCA9685Error init(uint16_t frequency = 50,
                  int sda_pin = 21,      // 默认 GPIO21
                  int scl_pin = 22,      // 默认 GPIO22
                  uint32_t i2c_speed = 100000);
```

### PCA9685Driver.cpp (第35-38行)
```cpp
i2c_conf.sda_io_num = static_cast<gpio_num_t>(sda_pin);  // GPIO21
i2c_conf.scl_io_num = static_cast<gpio_num_t>(scl_pin);  // GPIO22
i2c_conf.sda_pullup_en = GPIO_PULLUP_ENABLE;
i2c_conf.scl_pullup_en = GPIO_PULLUP_ENABLE;
```

| 通道 | 腿部 | 关节 | 功能 |
|------|------|------|------|
| 0 | 左前腿 | 髋侧摆 | 控制腿的左右摆动 |
| 1 | 左前腿 | 髋俯仰 | 控制大腿前后摆动 |
| 2 | 左前腿 | 膝俯仰 | 控制小腿弯曲 |
| 3 | 右前腿 | 髋侧摆 | 控制腿的左右摆动 |
| 4 | 右前腿 | 髋俯仰 | 控制大腿前后摆动 |
| 5 | 右前腿 | 膝俯仰 | 控制小腿弯曲 |
| 6 | 左后腿 | 髋侧摆 | 控制腿的左右摆动 |
| 7 | 左后腿 | 髋俯仰 | 控制大腿前后摆动 |
| 8 | 左后腿 | 膝俯仰 | 控制小腿弯曲 |
| 9 | 右后腿 | 髋侧摆 | 控制腿的左右摆动 |
| 10 | 右后腿 | 髋俯仰 | 控制大腿前后摆动 |
| 11 | 右后腿 | 膝俯仰 | 控制小腿弯曲 |

## 6. 电源系统

### 推荐电源配置
```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  USB 5V     │     │  DC-DC      │     │  舵机电源   │
│  或电池     │────►│  稳压模块   │────►│  (5V/2A+)   │
└─────────────┘     └─────────────┘     └─────────────┘
                                               │
                    ┌─────────────┐             │
                    │  PCA9685    │◄────────────┘
                    │  VCC/GND    │
                    └─────────────┘
```

### 电源要求
| 组件 | 电压 | 电流要求 |
|------|------|---------|
| ESP32 | 3.3V / 5V | 200-500mA |
| PCA9685 | 3.3V (逻辑) / 5V (舵机) | 取决于舵机 |
| 单个舵机 (SG90) | 5V | 100-250mA (堵转) |
| 单个舵机 (MG996R) | 5V | 500-900mA (堵转) |
| **12个舵机总计** | 5V | **2-5A (峰值)** |

## 7. 注意事项

### ⚠️ 重要警告

1. **I2C上拉电阻**
   - PCA9685模块通常已自带10K上拉电阻
   - 无需额外添加上拉电阻

2. **舵机供电**
   - ⛔ **禁止从ESP32的3.3V/5V引脚取电给舵机**
   - 必须使用独立的5V电源供电
   - 推荐使用5V/3A以上的开关电源

3. **共地连接**
   - ESP32、PCA9685、舵机电源**必须共地**
   - 不共地会导致I2C通信不稳定

4. **散热**
   - 12个舵机同时工作时PCA9685可能发热
   - 建议添加小型散热片

### 💡 调试建议

1. **首次测试**
   - 先只连接1-2个舵机测试
   - 确认I2C通信正常后再连接全部

2. **I2C地址确认**
   - PCA9685默认地址: 0x40
   - 可通过A0-A5引脚修改地址

3. **PWM频率**
   - 标准舵机: 50Hz (20ms周期)
   - 高速舵机: 100Hz
   - 数字舵机: 可达300Hz

## 8. 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 舵机不响应 | I2C通信失败 | 检查SDA/SCL接线 |
| 舵机抖动 | 电源不足 | 增大电源功率 |
| 舵机发热严重 | PWM频率不对 | 确认设置为50Hz |
| ESP32重启 | 电压跌落 | 检查电源和共地 |
| I2C找不到设备 | 地址错误 | 确认PCA9685地址为0x40 |
