
# 机器狗结构描述

## 1. 整体构型

* **类型**：典型四足机器人（Quadruped Robot）。
* **拓扑**：机身（刚体） + 四条对称分布的腿（每条腿 3 自由度）。
* **自由度总数**：12 个主动自由度（4 条腿 × 3 自由度/腿）。
* **腿式布局**：每条腿均为串联式关节链，基于机身侧壁安装，形成“机身–髋–膝–足端”的运动链。

---

## 2. 单条腿的关节与连杆结构

### （1）关节配置

* **第 1 关节：Hip Abduction/Adduction (HAA)**

  * 作用：控制腿的外展 / 内收。
  * 位置：机身与腿部的连接处，电机固定在机身上。
  * 特点：此关节的旋转会带动整条腿（包括后续所有连杆）围绕机身的垂直轴转动，实现腿的平面朝向调整。

* **第 2 关节：Hip Flexion/Extension (HFE)**

  * 作用：控制大腿前后摆动。
  * 安装：位于 HAA 之后，通过 **一个 90° 偏置连杆（第一连杆）** 与机身分离，保证电机安装空间及运动学合理性。
  * 驱动对象：通过连接结构，HFE 同时带动后续的 **第三连杆（大腿）** 和 **第四连杆（小腿/膝以下结构）**。

* **第 3 关节：Knee Flexion/Extension (KFE)**

  * 作用：控制膝关节的弯曲 / 伸展。
  * 驱动对象：直接作用于 **第四连杆（小腿）**，在 HFE 基础上提供额外相对角度，决定足端的高度与位置。

---

### （2）连杆构成

每条腿由 **4 个主要连杆**组成，但仅有 3 个主动自由度：

1. **连杆 1（Hip offset link）**

   * 直接与 HAA 相连，主要作用是将 HFE 的旋转轴从机身外偏置出来。
   * 与连杆 2 呈 **固定的 90° 夹角**。

2. **连杆 2（HFE link / thigh base）**

   * 与 HFE 直接耦合。
   * 为大腿部分的基底，连接到后续运动链。

3. **连杆 3（Thigh / upper leg）**

   * 由 HFE 驱动。
   * 构成膝关节的上部，连接到膝关节轴。

4. **连杆 4（Shank / lower leg）**

   * 同时受到 HFE 和 KFE 的驱动影响：

     * HFE 通过连杆传动耦合其部分运动。
     * KFE 直接调节该连杆的角度。
   * 末端连接 **足端（Foot）**，实现地面接触。

---

## 3. 运动学特性

* **耦合性**：HFE 的运动不仅作用于大腿（连杆 3），也传递到小腿（连杆 4），体现出一种“部分耦合驱动”。
* **解耦调节**：KFE 在 HFE 的基础上对小腿进行独立调节，实现精确的足端位置控制。
* **自由度与连杆数差异**：虽然腿部存在 4 个主要连杆，但实际只有 3 个驱动关节，自由度数小于连杆数，这在机器人学中属于 **欠驱动串联链**（underactuated serial chain）。

---

## 4. 足端与功能

* **足端（End-effector）**：位于第四连杆末端，通常为无驱动的被动接触点。
* **功能**：与地面相互作用，承受支撑反作用力，完成步态的落脚与支撑。
* **特性**：缺少踝关节（ankle），简化了机构设计，降低了复杂度和重量，但也限制了部分地形适应性。

---

## 5. 专业总结

该机器狗采用 **机身 + 四条 3 自由度腿**的结构形式。
每条腿通过 **HAA–HFE–KFE 的三关节配置**，配合 **4 连杆构型**实现运动。

* HAA 提供整腿的平面摆动；
* HFE 通过 90° 偏置后，驱动大腿及小腿的耦合运动；
* KFE 则在小腿处提供解耦控制，实现末端的精确定位。
