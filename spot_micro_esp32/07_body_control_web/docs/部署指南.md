# 部署指南

## 🚀 快速部署

本指南将帮助您快速部署ESP32四足机器人网页实时控制系统。

## 📋 环境准备

### 硬件要求

| 组件 | 规格 | 数量 | 说明 |
|------|------|------|------|
| ESP32开发板 | ESP32-DevKitC或兼容板 | 1 | 主控制器 |
| PCA9685 PWM驱动器 | 16通道12位PWM | 1 | 舵机驱动 |
| 舵机 | SG90或MG996R | 12 | 四足关节驱动 |
| 跳线 | 杜邦线 | 若干 | 连接线 |
| 电源 | 5V/3A适配器 | 1 | 舵机供电 |

### 软件要求

| 软件 | 版本 | 说明 |
|------|------|------|
| ESP-IDF | v4.4+ | ESP32开发框架 |
| Python | 3.6+ | ESP-IDF依赖 |
| Git | 最新版 | 代码管理 |
| 串口工具 | - | 调试监控 |

## 🔧 硬件连接

### ESP32与PCA9685连接

```
ESP32 GPIO21 (SDA) ──── PCA9685 SDA
ESP32 GPIO22 (SCL) ──── PCA9685 SCL
ESP32 3.3V ──────────── PCA9685 VCC
ESP32 GND ───────────── PCA9685 GND
```

### PCA9685与舵机连接

```
四足机器人舵机分配：
┌─────────────────────────────────┐
│  前左腿    │    前右腿          │
│  肩部: 0   │    肩部: 3         │
│  大腿: 1   │    大腿: 4         │
│  小腿: 2   │    小腿: 5         │
├─────────────────────────────────┤
│  后左腿    │    后右腿          │
│  肩部: 6   │    肩部: 9         │
│  大腿: 7   │    大腿: 10        │
│  小腿: 8   │    小腿: 11        │
└─────────────────────────────────┘
```

### 电源连接

```
5V电源正极 ──── PCA9685 V+
5V电源负极 ──── PCA9685 GND (与ESP32 GND共地)
```

## 💻 软件安装

### 1. 安装ESP-IDF

#### Linux/macOS
```bash
# 下载ESP-IDF
git clone --recursive https://github.com/espressif/esp-idf.git
cd esp-idf

# 安装工具链
./install.sh

# 设置环境变量
. ./export.sh
```

#### Windows
```cmd
# 下载并运行ESP-IDF安装器
# https://dl.espressif.com/dl/esp-idf-installer/

# 或使用命令行
git clone --recursive https://github.com/espressif/esp-idf.git
cd esp-idf
install.bat
export.bat
```

### 2. 验证安装

```bash
# 检查ESP-IDF版本
idf.py --version

# 检查工具链
xtensa-esp32-elf-gcc --version
```

## 📦 项目部署

### 1. 获取项目代码

```bash
# 克隆项目（如果从Git仓库）
git clone <project-repository>
cd web_control_robot

# 或直接使用现有代码目录
cd spot_micro_esp32/body_control_web
```

### 2. 配置项目

```bash
# 进入项目目录
cd spot_micro_esp32/body_control_web

# 配置项目
idf.py menuconfig
```

#### 重要配置项

**WiFi配置**
```
Component config → WiFi
- WiFi SSID: 您的WiFi名称
- WiFi Password: 您的WiFi密码
```

**串口配置**
```
Serial flasher config
- Flash size: 4MB
- Baud rate: 115200
```

**分区表配置**
```
Partition Table
- Partition table: Single factory app, no OTA
```

### 3. 修改WiFi参数

编辑 `main/pca9685Test.cpp`：

```cpp
// 修改WiFi配置
#define WIFI_SSID "Your_WiFi_Name"
#define WIFI_PASS "Your_WiFi_Password"
```

### 4. 编译项目

```bash
# 清理构建
idf.py fullclean

# 编译项目
idf.py build
```

### 5. 烧录固件

```bash
# 连接ESP32到电脑
# 查找串口设备
ls /dev/ttyUSB*  # Linux
ls /dev/cu.*     # macOS

# 烧录固件
idf.py -p /dev/ttyUSB0 flash

# 监控串口输出
idf.py -p /dev/ttyUSB0 monitor
```

## 🌐 网络配置

### 1. WiFi连接

启动ESP32后，查看串口输出：

```
I (2345) WIFI: WiFi连接成功
I (2346) WIFI: IP地址: 192.168.1.100
I (2347) WEB_ADAPTER: 🚀 Web server ready at http://192.168.1.100:12580
```

### 2. 静态IP配置（可选）

如需固定IP地址，修改 `pca9685Test.cpp`：

```cpp
// 配置静态IP
esp_netif_ip_info_t ip_info;
IP4_ADDR(&ip_info.ip, 192, 168, 1, 100);
IP4_ADDR(&ip_info.gw, 192, 168, 1, 1);
IP4_ADDR(&ip_info.netmask, 255, 255, 255, 0);
esp_netif_set_ip_info(netif, &ip_info);
```

### 3. 防火墙配置

确保路由器允许以下端口：
- **HTTP端口**: 12580
- **WebSocket端口**: 12580 (与HTTP共用)

# 解决方案：检查CMakeLists.txt中的REQUIRES

## 📈 监控和告警

### 1. 系统监控

```cpp
// 定期检查系统状态
void system_monitor_task(void *pvParameters) {
    while (1) {
        // 检查内存使用
        size_t free_heap = esp_get_free_heap_size();
        
        // 检查WiFi连接
        wifi_ap_record_t ap_info;
        esp_wifi_sta_get_ap_info(&ap_info);
        
        // 检查WebSocket连接数
        int active_connections = get_active_ws_connections();
        
        vTaskDelay(pdMS_TO_TICKS(30000)); // 30秒检查一次
    }
}
```

### 2. 告警机制

```cpp
// 内存不足告警
if (free_heap < 10000) {
    ESP_LOGW(TAG, "⚠️ 内存不足: %d bytes", free_heap);
    // 发送WebSocket告警消息
}

// 连接异常告警
if (wifi_disconnected_time > 60000) {
    ESP_LOGE(TAG, "❌ WiFi连接中断超过1分钟");
    // 尝试重连
}
```
