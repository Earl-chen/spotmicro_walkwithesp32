```python
@staticmethod
def cycloid_trajectory(phase: float, stride_length: float, step_height: float) -> tuple:
    """
    摆线轨迹（修正版 - 推荐用于 Walk 步态）
    
    特点：
    - 平滑的加速和减速
    - 适合舵机控制
    - 在摆动相和支撑相之间平滑过渡
    - **修正：X偏移对称，有向后蹬地动作**
    
    参数：
        phase: 相位 (0-1)
               0-0.5: 摆动相（抬腿向前摆动）
               0.5-1.0: 支撑相（向后蹬地）
        stride_length: 步长 (米)，默认 0.05 (50mm)
        step_height: 抬腿高度 (米)，默认 0.03 (30mm)
    
    返回：
        (x, z): 前后位置偏移, 上下位置偏移（米）
    
    修正要点：
    - X偏移范围：-stride_length/2 → +stride_length/2（对称）
    - phase=0: x = -stride_length/2（在最后面）
    - phase=0.5: x = +stride_length/2（在最前面）
    - phase=1.0: x = -stride_length/2（回到最后面）
    
    示例：
        >>> # 生成摆动相中点的轨迹
        >>> x, z = TrajectoryGenerator.cycloid_trajectory(0.25, 0.05, 0.03)
        >>> # x ≈ 0, z ≈ 0.015（最高点）
    """
    if phase < 0.5:  # 摆动相（抬腿向前摆动）
        t = phase * 2 * np.pi
        # X从 -stride_length/2 → +stride_length/2
        x = stride_length / 2 * (t / np.pi - 1)
        # Z抬起（摆线轨迹）
        z = step_height * (1 - np.cos(t)) / 2
    else:  # 支撑相（向后蹬地）
        t = (phase - 0.5) * 2 * np.pi
        # X从 +stride_length/2 → -stride_length/2
        x = stride_length / 2 * (1 - t / np.pi)
        # Z着地（高度为0）
        z = 0
    
    return x, z
```

**预期效果：**
- ✅ phase=0: x = -20mm, z = 0mm（着地点，在后面）
- ✅ phase=0.25: x = 0mm, z = 25mm（最高点，在中间）
- ✅ phase=0.5: x = +20mm, z = 0mm（着地点，在前面）
- ✅ phase=0.75: x = 0mm, z = 0mm（支撑相中点）
- ✅ phase=1.0: x = -20mm, z = 0mm（回到起点）
