## 坐标变换算法

### 4x4 齐次变换矩阵

**矩阵形式**
```
    ┌─────────────────────┐
    │  R(3x3)    t(3x1)  │
T = │  0 0 0      1      │
    └─────────────────────┘

其中:
  R = 旋转矩阵 (3x3)
  t = 平移向量 (3x1)
```

**位姿到矩阵**
```python
def _build_matrix(self):
    # 1. 计算旋转矩阵
    R = euler_to_matrix(self._roll, self._pitch, self._yaw)

    # 2. 构建 4x4 齐次矩阵
    T = np.eye(4, dtype=float)
    T[:3, :3] = R        # 左上 3x3 是旋转
    T[:3, 3] = self._pos # 右上 3x1 是平移
    self._T = T
```

**逆矩阵计算**
```python
def inverse_matrix(self):
    T = self.matrix()
    R = T[:3, :3]  # 旋转矩阵
    t = T[:3, 3]   # 平移向量

    # 逆矩阵: [ R^T  -R^T*t ]
    inv = np.eye(4)
    inv[:3, :3] = R.T       # 旋转矩阵的逆 = 转置
    inv[:3, 3] = -R.T @ t   # 平移向量变换
    return inv
```

**点坐标变换**
```python
# 机体坐标系 → 世界坐标系
def transform_point_body_to_world(self, p):
    p4 = np.array([p[0], p[1], p[2], 1.0])  # 齐次坐标
    w = self.matrix() @ p4                    # 矩阵乘法
    return w[:3]                              # 返回前3个元素
```

## 旋转矩阵算法

### 欧拉角到旋转矩阵

**旋转约定**
```
R = Rz(yaw) @ Ry(pitch) @ Rx(roll)
```

**基本旋转矩阵**
```python
# 绕 X 轴旋转 (横滚 Roll)
def rot_x(roll):
    c, s = cos(roll), sin(roll)
    return [[1,  0,   0 ],
            [0,  c,  -s ],
            [0,  s,   c ]]

# 绕 Y 轴旋转 (俯仰 Pitch)
def rot_y(pitch):
    c, s = cos(pitch), sin(pitch)
    return [[ c, 0,  s ],
            [ 0, 1,  0 ],
            [-s, 0,  c ]]

# 绕 Z 轴旋转 (偏航 Yaw)
def rot_z(yaw):
    c, s = cos(yaw), sin(yaw)
    return [[c, -s,  0],
            [s,  c,  0],
            [0,  0,  1]]
```

**组合旋转**
```python
def euler_to_matrix(roll, pitch, yaw):
    R_x = rot_x(roll)
    R_y = rot_y(pitch)
    R_z = rot_z(yaw)
    return R_z @ R_y @ R_x  # 矩阵乘法顺序很重要
```

### 旋转矩阵到欧拉角

**计算公式**
```python
def matrix_to_euler(R):
    # 1. 提取元素
    sy = sqrt(R[0,0]^2 + R[1,0]^2)

    # 2. 检查万向锁
    if sy < 1e-9:  # 奇点
        roll = atan2(-R[1,2], R[1,1])
        pitch = atan2(-R[2,0], sy)
        yaw = 0.0
    else:  # 正常情况
        roll = atan2(R[2,1], R[2,2])
        pitch = atan2(-R[2,0], sy)
        yaw = atan2(R[1,0], R[0,0])

    return roll, pitch, yaw
```

## 正运动学算法

### 算法流程

```
输入: 髋关节位置、关节角度 (hip_side, hip_pitch, knee)
输出: 关节位置序列 [hip, extension, hip_pitch, knee, foot]
```

### 几何结构

```
         髋关节 (hip)
            │
            │ l1_base = [0, ±l1, 0]
         ┌──┴──┐
         │ l1  │ ←── rot_x(hip_side)
         └──┬──┘
            │
      ┌─────┴─────┐
      │ extension │
      └─────┬─────┘
            │ l2_base = [0, 0, -l2]
         ┌──┴──┐
         │ l2  │
         └──┬──┘
            │
         ┌──┴──┐
         │ l3  │ ←── rot_y(hip_pitch)
         │ 大腿 │
         └──┬──┘
            │
         ┌──┴──┐
         │ l4  │ ←── rot_y(knee)
         │ 小腿 │
         └──┬──┘
            │
         脚部 (foot)
```

### 计算步骤

```python
def forward(self, hip_pos_body, joints):
    # 1. 髋侧摆旋转
    if is_left:
        l1_base = [0, +l1, 0]  # 左腿向外
    else:
        l1_base = [0, -l1, 0]  # 右腿向外

    rot_side = rot_x(joints.hip_side)
    extension = hip_pos_body + rot_side @ l1_base

    # 2. 髋俯仰段
    l2_base = [0, 0, -l2]
    hip_pitch = extension + rot_side @ l2_base

    # 3. 大腿段
    l3_base = [0, 0, -l3]
    rot_pitch = rot_y(joints.hip_pitch)
    knee = hip_pitch + (rot_side @ rot_pitch) @ l3_base

    # 4. 小腿段
    l4_base = [0, 0, -l4]
    rot_knee = rot_y(joints.knee_pitch)
    foot = knee + (rot_side @ rot_pitch @ rot_knee) @ l4_base

    return [hip_pos_body, extension, hip_pitch, knee, foot]
```

### 旋转矩阵组合

```
最终变换矩阵 = rot_side @ rot_pitch @ rot_knee

这个矩阵将小腿段从原始坐标系变换到髋坐标系
```

## 逆运动学算法

### 算法流程

```
输入: 脚部在髋坐标系中的位置 (x, y, z)
输出: 关节角度 (hip_side, hip_pitch, knee_pitch)
```

### 几何分析

```
侧视图 (YZ 平面)              正视图 (XZ 平面)

    Z ↑                        Z ↑
      │                          │
      │                          │
   (髋)●                        │
      │\                         │\
      │ \ l1                     │ \
      │  \                       │  \
      │   \                      │   \
      │    ● (extension)         │    ●
      │    │                     │    │
      │    │ l2                  │    │
      │    │                     │    │
      │    ● (hip_pitch)         │    ●
      │     \                    │     \
      │      \ l3                │      \
      │       \                  │       \
      │        ● (knee)          │        ●
      │         \                 │         \
      │          \ l4             │          \
      │           \               │           \
      │            ● (foot)       │            ●
      │            │               │            │
      └────────────┴────→ Y       └────────────┴────→ X

     H = √(y² + z²)              F = √(x² + (H-l1-l2)²)
     G = √(H² - l1²)             S = √(F² + x²)
```

### 计算步骤

```python
def inverse(self, foot_pos):
    x, y, z = foot_pos[0], foot_pos[1], foot_pos[2]

    # 1. 计算几何参数
    H = sqrt(y² + z²)           # 脚在 YZ 平面的投影距离

    # 2. 检查可达性
    if H < l1:
        return IKResult(False, None, "Target too close to hip")

    G = sqrt(H² - l1²)         # 去掉髋侧摆延伸后
    F = G - l2                  # 去掉髋俯仰段后
    S = sqrt(F² + x²)           # 到膝盖的距离

    if S > l3 + l4:
        return IKResult(False, None, "Target too far")

    # 3. 计算髋侧摆角度
    if is_left:
        hip_side = atan2(z, y) + acos(l1 / H)
    else:
        hip_side = π + atan2(z, y) - acos(l1 / H)

    # 4. 计算髋俯仰角度 (余弦定理)
    cos_hip = (S² + l3² - l4²) / (2 * S * l3)
    cos_hip = clamp(cos_hip, -1, 1)  # 数值稳定性
    hip_pitch = acos(cos_hip) - atan2(x, F)

    # 5. 计算膝俯仰角度 (余弦定理)
    cos_knee = (S² - l3² - l4²) / (2 * l3 * l4)
    cos_knee = clamp(cos_knee, -1, 1)
    knee_pitch = -acos(cos_knee)

    # 6. 角度范围约束
    hip_side = clamp(hip_side, -π/2, π/2)
    hip_pitch = clamp(hip_pitch, -π/2, π/2)
    knee_pitch = clamp(knee_pitch, -π, 0)

    return IKResult(True, LegJoints(hip_side, hip_pitch, knee_pitch), None)
```

### 余弦定理应用

```
在三角形 (l3, l4, S) 中:

l4² = l3² + S² - 2*l3*S*cos(π - hip_pitch)
cos_hip = (S² + l3² - l4²) / (2 * S * l3)

S² = l3² + l4² - 2*l3*l4*cos(π + knee_pitch)
cos_knee = (S² - l3² - l4²) / (2 * l3 * l4)
```

## 坐标系管理算法

### 坐标系树

```
world (根)
  │
  ├── body (机体)
  │     │
  │     ├── hip_left_front
  │     ├── hip_left_back
  │     ├── hip_right_front
  │     └── hip_right_back
  │
  └── ... (其他坐标系)
```

### 变换矩阵计算

```
目标: 计算 from_frame → to_frame 的变换

方法:
1. 计算 from_frame → world 的累积变换
   T_from_world = T_from_parent @ ... @ T_root_from

2. 计算 to_frame → world 的累积变换
   T_to_world = T_to_parent @ ... @ T_root_to

3. 计算 from_frame → to_frame
   T_from_to = inv(T_to_world) @ T_from_world
```

**代码实现**
```python
def get_transform_matrix(self, from_frame, to_frame):
    # 1. 累积到根节点
    mats_from = []
    cur = from_frame
    while cur is not None:
        node = self.nodes[cur]
        mats_from.append(node.rel.matrix())
        cur = node.parent

    # 2. 累积到根节点
    mats_to = []
    cur = to_frame
    while cur is not None:
        node = self.nodes[cur]
        mats_to.append(node.rel.matrix())
        cur = node.parent

    # 3. 计算到世界的变换
    T_from_world = np.eye(4)
    for m in mats_from:
        T_from_world = m @ T_from_world

    T_to_world = np.eye(4)
    for m in mats_to:
        T_to_world = m @ T_to_world

    # 4. 计算相对变换
    T_from_to = np.linalg.inv(T_to_world) @ T_from_world
    return T_from_to
```

## 脚步锁定算法

### 算法原理

```
目标: 机体位姿变化时，保持脚部世界坐标不变

步骤:
1. 捕获当前脚部世界坐标
2. 机体位姿变化
3. 对每条腿:
   a. 将脚部世界坐标转换到新的髋坐标系
   b. 使用逆运动学计算新的关节角度
```

### 代码实现

```python
# 1. 捕获脚步位置
def capture_fixed_feet(self):
    self.fixed_foot_world.clear()
    for leg_name in self.model.legs.keys():
        foot_world = self.model.get_leg_joints_world(leg_name)[-1].copy()
        self.fixed_foot_world[leg_name] = foot_world
    self.foot_lock_enabled = True

# 2. 机体位姿变化时自动更新关节角度
def set_body_pose(self, x, y, z, roll, pitch, yaw):
    # 更新机体坐标系
    body_node.rel.set_pose_euler(x, y, z, roll, pitch, yaw)

    # 如果启用脚步锁定，通过逆运动学保持脚步位置
    if self.foot_lock_enabled and self.fixed_foot_world:
        for leg_name, foot_world in self.fixed_foot_world.items():
            legkin, hip_frame = self.model.legs[leg_name]

            # 将脚部世界坐标转换到髋坐标系
            foot_in_hip = self.model.frame_manager.transform_point(
                foot_world, "world", hip_frame
            )

            # 逆运动学计算
            ikr = legkin.inverse(foot_in_hip)
            if ikr.success:
                self.model.update_joint_angles(leg_name, ikr.joints)
```

```
┌─────────────────────────────────────────────────────────────┐
│                    用户拖动滑块                              │
```

```python
# 问题: acos(x) 要求 x ∈ [-1, 1]
# 解决: 限制输入范围

cos_hip = max(-1.0, min(1.0, cos_hip))
```

## 性能优化

### 缓存机制

```python
# 问题: 重复计算坐标变换
# 解决: 缓存计算结果

def get_leg_joints_world(self, leg_name):
    # 检查缓存
    if leg_name in self._cache_world_joints:
        return self._cache_world_joints[leg_name]

    # 计算并缓存
    body_joints = self.get_leg_joints_body(leg_name)
    world_joints = [self.frame_manager.transform_point(
        p, self.body_frame, "world"
    ) for p in body_joints]
    self._cache_world_joints[leg_name] = world_joints

    return world_joints
```

### 缓存失效

```python
# 问题: 坐标系变化时，缓存失效
# 解决: 回调机制自动失效

def _setup_frame_callbacks(self):
    for node in self.frame_manager.nodes.values():
        if node.rel:
            # 设置变化回调
            node.rel.set_on_change(self.invalidate_cache)

def invalidate_cache(self, leg_name=None):
    if leg_name is None:
        self._cache_body_joints.clear()
        self._cache_world_joints.clear()
    else:
        self._cache_body_joints.pop(leg_name, None)
        self._cache_world_joints.pop(leg_name, None)
```
