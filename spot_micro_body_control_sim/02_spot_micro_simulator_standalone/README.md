# Spot Micro 独立模拟器

单文件完整的四足机器人模拟器，开箱即用。

## 特点

- **单文件实现** - 所有功能集成在一个 Python 文件中
- **18 参数控制** - 6 个机体位姿 + 12 个关节角度
- **脚步锁定** - 机体位姿变化时脚部世界坐标保持不变
- **实时可视化** - 世界坐标轴（红色）+ 机体坐标轴（蓝色）
- **零配置** - 无需安装额外模块，直接运行

## 快速开始

```bash
cd 02_spot_micro_simulator_standalone
python complete_quadruped_world_control.py
```

## 控制参数

### 机体位姿控制（6 个）
| 参数 | 范围 | 说明 |
|-----|------|------|
| X | -500 ~ 500 mm | 前后位置 |
| Y | -500 ~ 500 mm | 左右位置 |
| Z | -300 ~ 100 mm | 上下位置 |
| Roll | -45 ~ 45° | 翻滚角 |
| Pitch | -45 ~ 45° | 俯仰角 |
| Yaw | -180 ~ 180° | 偏航角 |

### 关节角度控制（12 个）
每条腿 3 个关节：
| 关节 | 范围 | 说明 |
|-----|------|------|
| 髋侧摆 | -90 ~ 90° | 腿部左右摆动 |
| 髋俯仰 | -90 ~ 90° | 大腿前后摆动 |
| 膝俯仰 | -180 ~ 0° | 小腿弯曲 |

## 机器人参数

```python
L1 = 60.5 mm     # 髋关节延伸段
L2 = 10.0 mm     # 髋关节垂直段
L3 = 111.126 mm  # 大腿长度
L4 = 118.5 mm    # 小腿长度
BODY_LENGTH = 207.5 mm  # 机体长度
BODY_WIDTH = 78 mm      # 机体宽度
```

## 功能说明

### 脚步锁定模式
- 拖动机体位姿滑块时，四只脚的世界坐标位置保持不变
- 系统自动通过逆运动学计算各腿关节角度
- 适合模拟机器人在不平整地面的姿态调整

### 状态监控
界面下方实时显示：
- 机体位姿状态
- 腿部关节角度
- 脚部世界坐标
