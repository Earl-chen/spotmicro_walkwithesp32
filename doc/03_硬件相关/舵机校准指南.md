# 舵机校准指南

## 概述

本文档描述如何使用 `01_test_pca9685_standalone` 项目对四足机器人的12个舵机进行独立校准。

## 为什么需要校准？

每个舵机都有不同的机械特性：
- **中位PWM值不同** - 范围在 615-685，而非统一的350！
- **行程范围不同** - 0°和180°对应的PWM值各异
- **安装位置影响** - 舵机安装角度会影响初始位置

## 硬件连接

```
ESP32 ←→ PCA9685
  GPIO21 (SDA) ←→ SDA
  GPIO22 (SCL) ←→ SCL
  3.3V          ←→ VCC
  GND           ←→ GND

PCA9685 舵机连接:
  通道 0-2  → 左前腿 (髋侧摆, 髋俯仰, 膝俯仰)
  通道 3-5  → 右前腿
  通道 6-8  → 左后腿
  通道 9-11 → 右后腿
```

## 编译与烧录

```bash
# 1. 加载ESP-IDF环境
source /home/cm/esp/v5.5/esp-idf/export.sh

# 2. 进入项目目录
cd spot_micro_esp32/01_test_pca9685_standalone

# 3. 编译
idf.py build

# 4. 烧录并监控
idf.py -p /dev/ttyUSB0 flash monitor
```

## 命令列表

### 基础命令

| 命令 | 格式 | 说明 |
|------|------|------|
| `s` | `s <0-11>` | 选择舵机 |
| `pwm` | `pwm <值>` | 设置PWM值 |
| `read` | `read [id]` | 读取当前PWM |
| `off` | `off` | 关闭所有舵机 |
| `status` | `status` | 显示系统状态 |

### 校准命令

| 命令 | 格式 | 说明 |
|------|------|------|
| `cal` | `cal` | 显示校准参数 |
| `test` | `test [id]` | 运行测试序列 |
| `scan` | `scan` | 扫描所有舵机 |
| `center` | `center` | 所有舵机回中位 |

### 快捷输入

直接输入PWM数值即可设置：
```
PCA9685[0]> 630    # 设置舵机0的PWM为630
```

## 校准流程

### 第一步：准备工作

1. **断开机械结构** - 首次校准时，建议断开舵机与机械结构的连接
2. **单独供电** - 确保舵机有独立的5V供电（不依赖ESP32的3.3V）
3. **打开串口终端** - 连接ESP32的USB串口

### 第二步：启动系统

```bash
idf.py -p /dev/ttyUSB0 flash monitor
```

系统启动后会自动将所有舵机设置到预设的中位。

### 第三步：校准单个舵机

```
# 1. 关闭所有舵机 (安全)
PCA9685[0]> off

# 2. 选择要校准的舵机
PCA9685[0]> s 0

# 3. 查看当前校准参数
PCA9685[0]> cal
舵机0: min=170, center=615, max=1060

# 4. 测试中位PWM
PCA9685[0]> 615

# 5. 如果舵机不在90°位置，手动调整PWM值
PCA9685[0]> 620
PCA9685[0]> 625
... 直到找到正确的中位

# 6. 测试限位
PCA9685[0]> 170    # 测试0°位置
PCA9685[0]> 1060   # 测试180°位置

# 7. 运行自动测试序列
PCA9685[0]> test
```

### 第四步：记录校准值

将新的校准值记录到以下位置：

1. **01_test_pca9685_standalone 项目**:
   - 文件: `main/drivers/test_pca9685_driver_refactored.cpp`
   - 数组: `SERVO_CALIBRATION[12]`

2. **06_body_control_uart 项目**:
   - 文件: `main/config/ConfigManager.cpp`
   - 函数: `ActualServoConfig::getActualServoConfigs()`

## 校准数据格式

```cpp
struct ServoCalibration {
    uint16_t min_pwm;    // 0度位置
    uint16_t center_pwm; // 90度位置 (中位)
    uint16_t max_pwm;    // 180度位置
};
```

## 常见问题

### Q: 舵机抖动怎么办？

1. 检查供电是否稳定（建议使用独立5V电源）
2. 检查PWM值是否在有效范围内
3. 检查机械结构是否卡住

### Q: 舵机不响应？

1. 检查I2C连接（SDA=GPIO21, SCL=GPIO22）
2. 检查PCA9685地址（默认0x40）
3. 运行 `scan` 命令检测连接状态

### Q: 舵机角度不对？

1. 使用 `cal` 查看当前校准值
2. 手动测试不同的PWM值找到正确位置
3. 更新校准数据到代码中

### Q: 如何退出监控？

按 `Ctrl + ]` 退出idf.py monitor

## 安全警告

⚠️ **重要提示:**

1. **不要使用350作为中位值！** 这是错误的！正确范围是615-685。
2. 首次测试时断开机械结构
3. 确保舵机有独立供电
4. 校准完成后及时更新代码中的校准数据
