# 步态测试程序 (Gait Test Standalone)

## 概述

本程序是一个专门用于测试四足机器狗步态算法正确性的命令行程序。它基于C++实现，与Python参考实现保持算法一致性，提供高精度的步态计算验证。

## 功能特性

### 🧪 测试类型
- **轨迹生成器测试**：验证摆线轨迹、椭圆轨迹等
- **Walk步态测试**：验证相位调度、轨迹生成
- **占空比验证**：确保25%摆动、75%支撑
- **支撑腿验证**：确保每时刻3条腿支撑
- **连续运行测试**：5秒连续运行验证

### 📊 测试覆盖
- **四条腿测试**：左前、右前、左后、右后腿
- **轨迹闭合性**：验证起点=终点
- **高精度验证**：位置误差通常在微米级别

### 📈 报告功能
- **实时结果显示**：测试过程中实时显示结果
- **详细统计信息**：成功率、误差分布、执行时间等

## 编译和运行

### 编译
```bash
# 使用提供的Makefile编译
make -f Makefile.gait_test

# 清理编译文件
make -f Makefile.gait_test clean
```

### 运行选项
```bash
# 快速验证测试
./gait_test --quick
./gait_test -q

# 批量预定义测试
./gait_test --batch
./gait_test -b

# 连续步态运行测试（5秒）
./gait_test --continuous
./gait_test -c

# 交互式菜单模式
./gait_test

# 显示帮助信息
./gait_test --help
./gait_test -h
```

## 技术特性

### 算法一致性
- 与Python参考实现完全一致的数学算法
- 修复了原始实现中的关键差异：
  - 轨迹闭合性问题（起点≠终点）
  - 支撑相公式错误

### 精度保证
- 使用单精度浮点数计算
- 实施数值稳定性保护
- 微米级别的计算精度

### 步态参数
- 步长：50mm（默认）
- 步高：30mm（默认）
- 步频：0.8Hz（默认）
- 占空比：0.75（Walk步态）

## 与 spot_micro_esp32 的关系

本模块是 **PC端步态算法验证工具**，与 `spot_micro_esp32/` 固件模块共享相同的核心步态代码。

### 📊 对比

| 对比项 | gait_test_standalone | spot_micro_esp32 |
|--------|---------------------|------------------|
| **运行环境** | PC（Linux） | ESP32芯片 |
| **依赖** | 纯C++标准库 | ESP-IDF框架 |
| **用途** | 算法验证 | 实机控制 |
| **编译速度** | 快（秒级） | 慢（分钟级） |
| **交互方式** | 命令行菜单 | Web/串口 |

### 🔗 代码对应关系

```
gait_test_standalone/                spot_micro_esp32/07_body_control_web/
├── gait/                            ├── main/gait/
│   ├── TrajectoryGenerator.hpp ──→ │   ├── TrajectoryGenerator.cpp (未来)
│   └── WalkGait.hpp            ──→ │   └── WalkGait.cpp         (未来)
```

### 🚀 开发流程

```
1. PC上开发/修改步态算法
        ↓
2. gait_test_standalone 验证算法正确性（轨迹闭合、占空比、支撑腿）
        ↓
3. 将验证通过的代码复制到 spot_micro_esp32
        ↓
4. 在ESP32上集成到控制系统中
```

### 💡 使用建议

- **修改步态算法前**：先在本模块验证，确保轨迹闭合、占空比正确
- **调试步态问题时**：优先在本模块排查，PC环境更易于调试
- **参数调整后**：运行批量测试确保所有验证通过
